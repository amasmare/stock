# 🌊 موقع ماسح إليوت الموجي (HTML — تشغيلي)

موقع **يعمل بالكامل في المتصفح** — بدون سيرفر، بدون بايثون، بدون تثبيت أي شيء.
ترفع ملفات CSV وتشوف الأنماط والرسم والـ Backtest مباشرة.

## التشغيل (3 طرق)

### 1) الأبسط — افتح الملف مباشرة
انقر مرتين على `index.html` → يفتح في متصفحك ويشتغل فوراً.

### 2) محلياً عبر سيرفر بسيط (اختياري)
```bash
cd web
python -m http.server 8777
# ثم افتح: http://localhost:8777
```

### 3) النشر على الإنترنت مجاناً (GitHub Pages)
1. ارفع مجلد المشروع على GitHub.
2. في المستودع: **Settings → Pages**.
3. تحت *Build and deployment* اختر **Branch: main** و **Folder: /web** (أو انقل ملفات
   `web/` إلى جذر المستودع واختر `/root`).
4. احفظ — خلال دقيقة يصير لك **رابط موقع عام** تشاركه مع أي أحد. 🎉

> لا يحتاج Streamlit ولا أي خدمة استضافة خاصة — صفحة ثابتة (static) تعمل على أي متصفح.

## كيف تستخدمه
1. اضغط **📂 رفع ملفات CSV** واختر ملف أو أكثر (كل ملف = سهم).
   - الأعمدة: `Date,Open,High,Low,Close,Volume` (يلزم High,Low,Close على الأقل).
2. **🔎 الماسح:** جدول مرتّب بالدرجة النهائية + تحميل CSV.
3. **📈 التفاصيل والرسم:** اختر سهماً → رسم الموجات + القوانين + التأكيدات + الأهداف.
4. **🧪 Backtest:** قياس جدوى الأنماط على الملفات المحمّلة.

## الملفات
| الملف | الدور |
|------|------|
| `index.html` | الواجهة |
| `elliott.js` | محرك الكشف (منقول من بايثون — نفس القوانين والتأكيدات) |
| `app.js` | ربط الواجهة + رسم الشارت (canvas) |
| `style.css` | التنسيق |

## من أين آتي بملفات CSV؟
استخدم سكربت `fetch.py` في المجلد الأب لتنزيل الأسعار من قائمة رموزك:
```bash
python fetch.py "tickers 2604.csv" --limit 100 --period 2y
```
ثم ارفع ملفات مجلد `data/` الناتجة في الموقع.
